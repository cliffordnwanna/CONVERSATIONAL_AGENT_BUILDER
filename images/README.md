# Images and Assets

This folder contains screenshots and images for the Conversational Agent Builder project.

## Screenshot Guidelines

### Recommended Sizes
- **Hero screenshots**: 1200x630px (16:9 ratio)
- **Mobile screenshots**: 375x667px (iPhone X ratio)
- **Feature screenshots**: 800x600px (4:3 ratio)

### File Naming Convention
- `hero-homepage.png` - Main landing page
- `mobile-homepage.png` - Mobile view of homepage
- `builder-interface.png` - Agent builder interface
- `analytics-dashboard.png` - Analytics dashboard
- `deployment-page.png` - Deployment interface
